# Resumo Executivo

O projeto `convite` contém a segunda versão funcional do convite web de casamento para Douglas e Regiane, criada a partir de `prompt.md` e evoluída conforme `prompt2.md`.

A V2 mantém a estrutura aprovada, mas aquece o dark com champagne, dourado queimado discreto, off-white, luz difusa e transições mais fluidas. As fotos reais aparecem no hero, na galeria e no encerramento. A história recebeu marco visual para `12/05/2020`, os cards ficaram menos secos e a timeline foi refinada com linha animada e pontos com glow.

Não há RSVP, formulário, botão de WhatsApp ou coleta de dados. Os links dos mapas seguem como `#` e devem ser substituídos antes da publicação.

Validação realizada em 2026-06-14: `npm run build` e `npm audit`, ambos aprovados.

O projeto também foi preparado para Vercel com `vercel.json`, `.nvmrc` e engine Node `>=22.12.0`.
