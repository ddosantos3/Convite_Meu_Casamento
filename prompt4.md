# prompt4.md — Aplicar identidade visual final no convite web

Jarvis, agora precisamos evoluir o convite web de Douglas & Regiane para a identidade visual final aprovada.

A versão atual tem a estrutura correta, mas ainda está com aparência de página web simples, estática, escura demais e com cara de template/IA. O objetivo agora é transformar o projeto em um convite digital realmente emocional, vivo, elegante, luminoso e com identidade própria.

Não quero uma landing page comum.
Não quero aparência de WordPress.
Não quero visual genérico.
Não quero uma página escura com textos empilhados.

Quero um convite web conceitual, fluido, humano e visualmente memorável.

---

## 1. Regra principal

Use somente o que já existe no projeto.

As fotos já estão salvas na pasta do projeto. Use essas fotos reais. Não invente imagens, não use banco de imagens, não coloque placeholders genéricos se já houver fotos reais disponíveis.

Não invente novos fatos sobre a história.
Não invente novos textos sem necessidade.
Não altere o sentido emocional já definido.

O trabalho agora é aplicar direção visual, fluidez, animação, decoração e identidade.

## 10. Glassmorphism e glowmorphism

Os cards precisam ter aparência real de vidro e brilho, não apenas fundo cinza.

Usar algo próximo:

```css
background: rgba(255, 255, 255, 0.055);
backdrop-filter: blur(18px);
border: 1px solid rgba(255, 255, 255, 0.12);
box-shadow:
  0 20px 60px rgba(0, 0, 0, 0.35),
  inset 0 1px 0 rgba(255, 255, 255, 0.08);
```

Para glow sutil:

```css
box-shadow:
  0 0 0 1px rgba(212, 175, 112, 0.14),
  0 0 40px rgba(212, 175, 112, 0.08);
```

Usar glow em:

* nomes do casal;
* badge da data;
* pontos da timeline;
* ícones;
* bordas de cards importantes;
* botões de localização;
* divisórias;
* detalhes decorativos.

Não aplicar glow em tudo.
O brilho precisa conduzir o olhar.
